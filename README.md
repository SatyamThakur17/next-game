# NEXT™  
### A Game of Inconvenience

> A corporate onboarding experience that slowly turns against you.

NEXT™ is a browser-based psychological chaos experiment disguised as a clean SaaS interface.

There is only one button.

It gets worse.

You cannot win.

## 🧠 Concept

NEXT™ is built around one simple idea:

> Progress should feel inconvenient.

Every time you advance a level:
- The interface becomes slightly more hostile
- The UI introduces permanent friction
- The system pretends to help you

By Level 10:
- You reach the "Final Level"
- You unlock "Hard Mode™"
- All inconveniences stack permanently

The game never truly ends.

## ⚙️ Features

- 🖱 Evasive button that dodges your cursor
- ⏳ Hold-to-confirm interactions
